# 🔮 AI Fortune Teller Web App

An interactive web application that simulates fortune-telling experiences such as tarot readings and face-based predictions through dynamic user interfaces.

---

## Overview

This project is a front-end web application designed to explore how AI-like experiences can be created through interactive UI and logic-based systems.

Users can engage with different features (tarot reading, face analysis) to receive personalized and entertaining “fortune” outputs.

The project focuses on **user experience, interactivity, and creative application design**, inspired by AI-driven products.

---

## Features

- 🔮 Tarot card reading interface  
- 👤 Face reading simulation  
- 🎨 Interactive multi-page UI  
- ⚡ Real-time response generation using JavaScript  
- 💡 Multiple fortune-telling experiences in one app  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## How It Works

1. User selects a feature (tarot or face reading)  
2. JavaScript processes user interaction  
3. Predefined logic generates a fortune or prediction  
4. The result is dynamically displayed in the UI  

---

## Example Experience

- Select a tarot card → receive a personalized fortune  
- Use face reading → get personality/fortune insights  
- Navigate between different interactive pages  

---

## Purpose

This project was built to explore:

- How AI-like interactions can be simulated without full ML models  
- Designing engaging and creative user experiences  
- Building interactive web applications using JavaScript
